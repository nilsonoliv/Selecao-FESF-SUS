## 🚀 Projetos em Destaque

### 1. Dashboard Estratégico - Aprovação TI Bancária
Plataforma EdTech estilo *Single Page Application* projetada para gerenciar rotinas complexas de estudo através de análise de dados e engenharia reversa estatística.
* **Soluções Arquiteturais Aplicadas:** Implementação do padrão *Single Source of Truth* (SSOT) para gerenciamento de estado global mutável, desacoplando completamente os bancos de dados do edital das funções de renderização da UI.
* **Integração de Nuvem Dinâmica:** Uso de expressões IIFE assíncronas aliadas a *Dynamic Imports* para injetar o Firebase SDK modularizado de forma isolada, garantindo autenticação segura (*Auth Before Query*).
* **Stack Tecnológica:** Vanilla JavaScript (ES6+), Firebase Firestore & Auth, Chart.js para visualização de métricas e CSS estruturado via Design Tokens.

### 2. Dashboard Tático - Operação CRECI-BA
Web app tático e gamificado projetado em modelo PWA para otimização e manutenção de consistência em rotinas de alta performance.
* **Soluções Arquiteturais Aplicadas:** Arquitetura estritamente modular baseada em *Module Pattern* (IIFE) para total isolamento de escopo. Aplicação do *Wrapper Pattern* para abstração da API nativa do `localStorage`, blindando a aplicação contra acoplamentos e facilitando futuras migrações de banco de dados.
* **Stack Tecnológica:** HTML5 Semântico, CSS Grid/Flexbox, Vanilla JS, Service Workers e persistência/backup estruturada em arquivos `.json`.

### 3. Ambiente de Desenvolvimento Full Stack (Em Consolidação)
Estrutura inicializada com ferramentas modernas voltada para a criação de microsserviços integrados de alta disponibilidade.
* **Stack Tecnológica:** React, TypeScript, Vite e ESLint configurado para Type-Aware Linting direcionado à produção.
